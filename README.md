## ğŸš€ Getting Started

### 1. Prerequisites

- Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
- Install the [Expo Go](https://expo.dev/go) app on your mobile device or use an emulator.

### 2. Installation

Clone the repository and install dependencies:

```bash
npm install
```

### 3. Environment Variables

Create a `.env` file in the root directory and add your API keys:

```bash
EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_key
EXPO_PUBLIC_GEMINI_API_KEY=your_gemini_key
EXPO_PUBLIC_FATSECRET_CLIENT_ID=your_fatsecret_id
EXPO_PUBLIC_FATSECRET_CLIENT_SECRET=your_fatsecret_secret

# Firebase Config
EXPO_PUBLIC_FIREBASE_API_KEY=your_fb_key
EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN=your_fb_domain
EXPO_PUBLIC_FIREBASE_PROJECT_ID=your_fb_project_id
EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=your_fb_bucket
EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_fb_sender_id
EXPO_PUBLIC_FIREBASE_APP_ID=your_fb_app_id
```

### 4. Running the App

Start the development server:

```bash
npx expo start
```

Scan the QR code with your Expo Go app to see the app in action!
